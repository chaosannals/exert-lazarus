## 插件

菜单 >  软件包  >  插件管理

由于 Lazarus 界面是拆成一堆小窗口的，所以需要安装插件合并成一个大窗口。

AnchorDocking
AnchorDockingDsgn
DockedFormEditor

### 通过 lpk 文件安装

菜单 >  软件包  >  打开 lpk
